# Car Rental System (Assignment 2)

Бұл жоба Java-дағы ООП принциптерін (1-4 лекция) көрсету үшін жасалған.

## Жобаның құрылымы:
* **Vehicle (Abstract)** - базалық класс (Abstraction & Encapsulation).
* **Car** - мұрагер класс (Inheritance & Polymorphism).
* **Client & RentalService** - көмекші кластар.
* **Main** - деректерді өңдеу, сұрыптау (Sorting) және нәтиже шығару.

## Тақырыптар:
1. Classes & Objects
2. Encapsulation (Private fields, Getters/Setters)
3. Inheritance (extends, super)
4. Polymorphism (Abstract methods, Override)
